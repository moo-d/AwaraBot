## üöÄ Installation

### Prerequisites
- Go 1.20+
- Node.js 1.18+
- WhatsApp mobile app

### Setup
1. Clone the repository
   ```bash
   git clone https://github.com/moo-d/AwaraBot
   cd AwaraBot
   ```

2. Install dependencies
   ```bash
   go run tidy
   npm install
   ```

3. Configure environment
   ```bash
   cp .env.example .env
   nano .env  # Edit with your config
   ```

4. Build and run
   ```bash
   npm run prod
   ```
